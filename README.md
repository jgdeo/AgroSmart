# 🌱 AgroSmart

> O aplicativo que leva a tecnologia ao pequeno agricultor.

AgroSmart é uma plataforma web de monitoramento agrícola em tempo real, desenvolvida para ajudar pequenos produtores a acompanhar as condições do campo e manter conformidade com a legislação ambiental.

---

## 📋 Funcionalidades

- **Página institucional** com informações sobre leis ambientais na agricultura
- **Painel de monitoramento em tempo real** com sensores simulados
- 🌡️ Sensor de **temperatura do ar**
- 💧 Sensor de **umidade do solo**
- 🌫️ **Detector de fumaça** com alerta de incêndio
- Cenários de simulação: Dia Normal, Seca Intensa, Pós-Chuva e Incêndio
- Histórico de leituras com status visual
- Alertas automáticos com recomendações

---

## 🚀 Como usar

### Opção 1 — Abrir direto no navegador

Basta abrir o arquivo `index.html` no seu navegador. Nenhuma instalação necessária.

### Opção 2 — GitHub Pages (recomendado)

1. Faça o fork ou clone este repositório
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `main` e pasta `/ (root)`
4. Clique em **Save**
5. Acesse `https://seu-usuario.github.io/agrosmart`

### Opção 3 — Deploy no Vercel

```bash
npm install -g vercel
vercel
```

Ou arraste a pasta para [vercel.com/new](https://vercel.com/new).
